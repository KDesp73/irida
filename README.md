# Chess Engine Project

## Quick Start

### Prerequisites

- `gcc` with support for `-fsanitize`
- `make`
- `bear` (optional, for generating `compile_commands.json`)
- `zig`
- `linux`

### Building the Project

```bash
git clone https://github.com/KDesp73/chess-engine --recursive && cd chess-engine
make -C castro all
zig build
````
