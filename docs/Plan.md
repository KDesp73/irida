# Φάση 1 – Βασική Υλοποίηση (Απαραίτητα)

Στόχος: Δημιουργία μιας πλήρως λειτουργικής σκακιστικής μηχανής με όλα τα βασικά στοιχεία.

Περιλαμβάνει:

- Γεννήτρια κινήσεων (Move Generation):
- Αξιολόγηση θέσης (Evaluation):
- Αναζήτηση (Search):
  Αλγόριθμος Negamax ή Minimax με Alpha-Beta Pruning και Iterative Deepening.
- Διαχείριση βιβλίων κινήσεων (Opening Book):
- Αποθήκευση θέσης (Transposition Table):
  Χρήση Zobrist hashing για αναγνώριση επαναλαμβανόμενων θέσεων.
- UCI/Χειρισμός διεπαφής (Interface):
  Επικοινωνία με GUI ή άλλα προγράμματα για testing.

# Φάση 2 – Βελτιώσεις (Προαιρετικά αλλά Σημαντικά)

Στόχος: Βελτίωση ποιότητας παιχνιδιού και ταχύτητας.

Περιλαμβάνει:

- NNUE Evaluation:
  Υλοποίηση νευρωνικού δικτύου για ταχύτερη και πιο ακριβή αξιολόγηση θέσης. (Χρήση προϋπολογισμένων βαρών)
- Move Ordering / Killer Moves / PV Search:
  Βελτίωση του search για μεγαλύτερο βάθος αναζήτησης σε λιγότερο χρόνο.
- Quiescence Search:
  Αποφυγή horizon effect.
- Late Move Reductions, Null Move Pruning:
  Επιπλέον τεχνικές pruning για ταχύτερη αναζήτηση.
- Βελτιώσεις στην αποθήκευση θέσεων και hashing.
- Syzygy

# Φάση 3 – Προαιρετικά / Προχωρημένα

Στόχος: Προχωρημένες δυνατότητες που δεν είναι απαραίτητες για βασική λειτουργία αλλά προσθέτουν ενδιαφέρον.

Περιλαμβάνει:

- Εκπαίδευση custom νευρωνικού δικτύου:
- Ανάλυση παιχνιδιών / Chess Engine GUI integration:
  Δημιουργία εργαλείων για ανάλυση παρτίδων, ενσωμάτωση στο GUI.
- Monte Carlo Tree Search ή άλλα search paradigms (πέρα από Negamax).
- Στατιστικές και analytics:
  Αξιολόγηση απόδοσης του engine, σύγκριση με άλλα engines.
- LLM integration: 
  Χρήση της engine απο open source LLM για εκπαιδευτικούς σκοπούς
